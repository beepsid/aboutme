001 : BPO Call Center Analytics

A data analytics project simulating a Business Process Outsourcing (BPO) call center environment. Covers synthetic data generation (50,000 call records), a normalized MySQL schema, SQL-based KPI analysis, statistical anomaly detection using z-scores, and a Power BI dashboard.

Stack: Python · MySQL · SQL · Power BI

Highlights: 17 KPI queries using window functions and subqueries · Injected anomaly (Tech Support drop rate spike) detected via z-score analysis · Rolling 3-week average drop rate via SQL window functions
